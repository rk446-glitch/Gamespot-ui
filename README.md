# Gamespot-ui
This project is a front-end gaming e-commerce user interface developed completely from scratch using HTML5 and CSS3. 
GameSpot is a front-end web project developed using HTML5 and CSS3, focused on designing a clean and professional gaming e-commerce user interface.

This project was built entirely from scratch, without using templates, frameworks, or UI libraries.
The design follows real-world e-commerce layout patterns while maintaining original structure and styling.

The primary objective of this project was to practice layout design, Flexbox, and UI structuring by manually implementing each component.
